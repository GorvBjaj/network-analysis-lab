# Virtual Machine Setup

This lab uses two separate virtual machines to simulate a controlled network environment.

## Virtual Machines Used
- Kali Linux (Attacker)
- Ubuntu (Victim)

## Setup Approach
- Default installation options were used for both operating systems.
- No additional services were enabled during installation.

## Purpose
- Kali Linux was used to generate network-based attacks.
- Ubuntu was used to observe, capture, and analyze network traffic.
