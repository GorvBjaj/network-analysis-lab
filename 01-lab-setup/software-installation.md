# Software Installation

This section documents the essential software installed on the virtual machines before network isolation.

## Ubuntu (Victim Machine)

The following tools were installed on Ubuntu to support network analysis:

### Wireshark
- Used to capture and analyze network traffic.
- Installed before isolating the network to ensure proper packet capture functionality.

### NetworkManager CLI (nmcli)
- Used to configure static IP addresses.
- Required for manual network configuration after isolation.

## Kali Linux (Attacker Machine)

- Kali Linux comes with required attack and scanning tools pre-installed.
- No additional software was installed for this lab.
