# TCP SYN Attack

This section documents a TCP SYN attack performed in a controlled lab environment.

## Objective
To observe how a TCP SYN flood appears at the packet level and how it affects the target system.

## Lab Roles
- Attacker: Kali Linux (192.168.56.10)
- Victim: Ubuntu (192.168.56.20)

## Activity Overview
- A TCP SYN attack was launched from Kali Linux.
- Network traffic was captured on Ubuntu using Wireshark.
- Packet behavior was analyzed to identify SYN flood patterns.

## Evidence
- PCAP files containing captured attack traffic
- Wireshark analysis screenshots

## Notes
This attack was conducted only within an isolated internal network for educational purposes.
