# ARP Spoofing Attack

This folder documents an ARP spoofing attack performed in an isolated lab environment.

## Objective
To observe and analyze malicious ARP traffic and understand its impact from a SOC analyst perspective.

## Lab Details
- Attacker: Kali Linux (192.168.56.10)
- Victim: Ubuntu (192.168.56.20)
- Network: Isolated internal network

## Activity Summary
- An ARP spoofing attack was launched from the attacker machine.
- Network traffic was captured on the victim using Wireshark.
- ARP packet behavior was analyzed to identify spoofing activity.

## Evidence
- PCAP files containing captured ARP traffic
- Screenshots of packet analysis

## Notes
This activity was conducted strictly for educational purposes within a controlled lab.
